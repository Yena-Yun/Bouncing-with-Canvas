Vanilla 자바스크립트로 <BR/>
Canvas 공식 문서의 공 튕기기 애니메이션에 <BR/>
사용자가 조작할 수 있는 버튼 UI를 추가해보았습니다.

## 추가한 기능
1. **애니메이션 시작하기**
2. **일시중지 및 다시 시작하기**
3. **공 복제하기 (공의 갯수를 거듭제곱수로 복제)**
4. **애니메이션 멈추기**
